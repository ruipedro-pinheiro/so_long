# SO_LONG
## Description
	This project is all about learning to do visual programs using a lib for x-windows on linux. More specifically, i must build a game made from C using minilibX, we must have an exit, one player, one consumable/item. And most importantly a parser of maps, and make sure they are valid following all requirements (flood fill algorithm per example)
